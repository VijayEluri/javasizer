Use this library to estimate the size of a java object graph.  The library will traverse the graph using reflection and use JVM instrumentation to estimate the size of the Objects in your JVM.

build:
	mvn compile
	you want: target/sizer-jar-with-dependencies.jar

usage:
	add jar to your classpath
	
in your code, do this to get a deepSize in bytes of a java object:
	org.jh.Sizer.sizeof(object)
